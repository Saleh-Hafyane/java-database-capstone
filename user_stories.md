# Smart Clinic User Stories

## Admin User Stories
1. As an admin, I want to add new doctor profiles to the system so that patients can see available specialists.
2. As an admin, I want to manage user roles and permissions to ensure data security.
3. As an admin, I want to view a dashboard of all clinic appointments to monitor daily operations.
4. As an admin, I want to generate reports on patient registrations to track clinic growth.
5. As an admin, I want to update clinic contact information so patients can reach us easily.

## Patient User Stories
1. As a patient, I want to view a list of doctors without logging in to explore my options.
2. As a patient, I want to sign up with my email and password to book future appointments.
3. As a patient, I want to log into the portal to manage my existing bookings.
4. As a patient, I want to book an hour-long appointment to consult with a specific doctor.
5. As a patient, I want to view my upcoming appointments to prepare for my visits.

## Doctor User Stories
1. As a doctor, I want to log into the portal to manage my daily appointment schedule.
2. As a doctor, I want to view my appointment calendar to stay organized.
3. As a doctor, I want to mark specific slots as unavailable to inform patients of my schedule.
4. As a doctor, I want to update my specialization and contact info to keep patients informed.
5. As a doctor, I want to view patient details before an appointment to provide better care.
